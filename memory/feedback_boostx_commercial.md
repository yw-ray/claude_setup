BoostX-SDS(`/home/youngwoo.jeong/grepo/260406_BoostX-SDS`) 연구 방향은 **상용 제품으로 출시되는 것을 전제**로 스코핑한다. "팔리는 것"이 1차 기준이고, 논문 novelty·학술적 참신성은 2차 기준이다. 사용자의 PhD 연구 주제(EDA timing closure + ML, Austin Rovinski)와는 **분리된 별개의 track**으로 다룬다.

**Why:** 사용자가 명시적으로 정정했음. BoostX-SDS는 MangoBoost의 상용 SDS 제품이며, 이 위에 쌓는 "혁신"은 구매 고객이 인식할 수 있는 차별화여야 함. PhD 주제와 연결 짓는 프레이밍은 사용자 의도와 어긋남.

**How to apply:**
- 연구 방향 제안 시 기준: (1) 구매자가 누구인가, (2) 경쟁사(VAST, WekaIO, Pure, NetApp, DDN 등) 대비 측정 가능한 우위, (3) MangoBoost DPU 자산 활용, (4) 세일즈 사이클에 맞는 PoC 난이도.
- "PhD 관심사와 정합" 같은 프레이밍 사용 금지. 학술 venue(FAST/OSDI/ATC) 정합은 secondary, primary는 reference customer 확보 가능성.
- Thesis/novelty는 여전히 중요하지만, "논문이 되는가"보다 "제품 차별화 스토리가 되는가"로 평가.
- 연구 산출물 형태: 논문뿐 아니라 benchmark report, reference architecture, whitepaper, customer PoC 결과가 primary artifact일 수 있음.
