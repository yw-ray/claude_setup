# Mango CLI 프로젝트 - 프로젝트별 규칙

## 프로젝트 개요
- Mango 프로젝트를 위한 명령줄 인터페이스 도구

## 코드 스타일
- 인덴테이션: 4 spaces

## 프로젝트 구조
- CLI 명령어는 모듈화하고 재사용 가능하게 작성
- 비즈니스 로직과 CLI 인터페이스 분리

## 문서화
- 모든 명령어에 명확한 도움말 메시지 추가
